## About me

I'm a Python web developer passionate about code performance and developer experience. While I work primarily with Python, I also explore Cython, C++, and Go for personal projects and optimization experiments.

For me, programming is more than a job—it's a hobby. I enjoy open source, improving the tools and languages I use, and contributing to a better developer experience. I'm obsessed with performance tuning, reducing unnecessary complexity, and eliminating resource waste. If something can be leaner or faster, I chase it.

I’m interested in high-load systems and asynchronous architectures, though I'm still waiting for my first >1 RPS system. 

I value AI as a tool— like linters or IntelliSense— but I'm skeptical of AI agents and similar trends. I build for humans, and I want to make code better for developers everywhere.
